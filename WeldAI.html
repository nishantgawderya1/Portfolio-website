Weld AI

In Progress — YOLOv8 baseline complete

Weld AI is an end-to-end automated defect recognition (ADR) system for radiographic weld inspection. Built on the RIAWELC dataset — 24,000 industrial X-ray images across 4 defect classes (crack, porosity, slag inclusion, and no-defect) — the system replicates and automates the work of a certified NDT inspector using deep learning.

Pipeline:

  • X-ray Upload — raw radiographic image input
  • Preprocessing — noise reduction, contrast normalisation, resizing
  • YOLOv8 Detection — bounding box localisation of defect regions across 4 classes
  • U-Net Segmentation — pixel-level mask generation for each detected defect
  • Pixel-to-mm Calibration — converts pixel measurements to real-world dimensions
  • ASME Report Generation — structured inspection report output as PDF

Tech Stack:

YOLOv8 (Ultralytics) U-Net (PyTorch) FastAPI MLflow Docker DVC ONNX

Key Metrics:

  • mAP@0.5: 0.87
  • F1 Score: 0.93
  • Inference: <1.8s
  • Dataset: 24K images
