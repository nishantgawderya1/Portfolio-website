ReVive

Pre-Incubated @ IIT Delhi (IHFC)

ReVive is an AI-driven physiotherapy platform developed under the IHFC ecosystem at IIT Delhi — a national mission hub for collaborative robotics and human-machine interaction. It eliminates the need for wearable hardware by using real-time computer vision to guide patients through physiotherapy exercises with precision matching trained physiotherapist standards.

Key Features:

  • Real-time human pose estimation pipeline tracking 33 anatomical keypoints via MediaPipe Pose for accurate body movement analysis.
  • Joint angle computation module with ±3° tolerance threshold, delivering posture feedback on par with trained physiotherapists.
  • Low-latency WebSocket communication layer achieving sub-200ms round-trip feedback for seamless real-time guidance.
  • Exploratory Data Analysis (EDA) on pose datasets with noise filtering and landmark normalization for model stability across diverse conditions.
  • No wearable sensors needed — purely vision-based, making physiotherapy accessible and hardware-free.

Technologies Used:

Python MediaPipe Pose OpenCV WebSockets Computer Vision NumPy
